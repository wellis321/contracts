## The 11 Tender Statuses Explained

Think of it like a job application process:

1. **Market Engagement** = Heard about the job, initial chat
2. **Pre-Qualification** = Submitting your CV/application
3. **Tender Submitted** = Full application sent, waiting
4. **Under Evaluation** = They're reviewing your application
5. **Clarification Requested** = They have questions
6. **Awarded** = You got the job! ğŸ‰
7. **Lost** = Someone else got it
8. **Contract Live** = You're working there now
9. **Extension Negotiation** = Discussing staying longer
10. **Retender Pending** = Contract ending, reapplying needed
11. **Contract Ended** = No longer working there

**You don't need to use all of them** - just the ones that apply to your situation.

**A:** No! The page **organizes** them so you only see what matters:
- Most will be "Contract Live" (just running normally)

### Q: "Do I need to update statuses constantly?"
**A:** No! Only when things change:
- Submit a tender â†’ Set to "Tender Submitted"
- Win a contract â†’ Set to "Awarded" then "Contract Live"
- Contract nearing end â†’ Set to "Extension Negotiation" or "Retender Pending"

### Q: "What if I don't know the tender status?"
